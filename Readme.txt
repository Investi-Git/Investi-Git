Bienvenue dans le jeu Investi-Git, un jeu d'enquête sur le logiciel de versionning Fork.

Un meurtre à été commis dans la ville de Greenfield, le PDG de Urban One, Patrick Johnson est mort le 15 janvier. Votre objectif est de retrouver qui est le / la meurtrier/ère.

Une fois le coupable trouvé vous devrez marquer le prénom du coupable dans le fichier Solution.

Nomenclature du tueur :

exemple : Harry
exemple : Stephanie

Attention, la majuscule est nécessaire.

Dans le logiciel fork, chaque branche est un jour de l'histoire, pour y acceder, il suffit de double cliquer sur la branche de votre choix et de selectionner Checkout. Votre dossier se mettra ensuite a jour.

Il est conseillé de jouer a plusieurs et de vous munir d'un stylo et de feuilles.

Bonne enquête !
